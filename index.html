Paste from Excel

Инструкция:
1. Откройте ваш Excel файл (например PACKING LIST)
2. Выделите строку с заголовками + все строки с данными
3. Нажмите Ctrl+C (скопировать)
4. Кликните в поле ниже и нажмите Ctrl+V (вставить)
5. Нажмите Добавить в список
Программа автоматически найдёт колонки: Description / Название Qty / Cartons Weight (kg) Size / L x W x H
